## 新メカニクス：トリンケット・グレーディングとジェム

**brain rot evolution ライブイベント**に備えて、開発者はトリンケット（装飾品）の「グレード」システムを導入しました。このシステムにより、プレイヤーは装備品のステータス倍率を上げるためにロール（再抽選）を行うことができます。これは膨大なジェム消費先（ジェムシンク）となっており、高ティアのロールには1回あたり100億ジェム以上のコストがかかることもあります。

グレードはFからS+まであり、各ティアごとにダメージ、経験値獲得量、または運（ラック）に段階的なブーストが付与されます。Flush RoomsやBrainrot Heavenのような終盤エリアで足止めを食らっているプレイヤーにとって、新しいボスにダメージを与えるためには、少なくともAグレードのトリンケットを確保することが必須条件になりつつあります。

### トリンケット・グレード倍率

| グレード | 推定ステータスブースト | レア度 / 難易度 |
|---|---|---|
| **Fグレード** | +1-2% | 非常に一般的 |
| **Dグレード** | +5% | 一般的 |
| **Bグレード** | +15% | アンコモン |
| **Aグレード** | +30% | レア |
| **Sグレード** | +50% | ベリーレア |
| **S+グレード** | +75% 以上 | レジェンダリー |

プレイヤーからの報告によると、S+グレードのRNG（乱数）は特に過酷です。トリンケットのグレードを最大化する前に、大量のチケットを使用してTrololeroホイールからジェムを蓄えておくことをお勧めします。

2. **Mount Balaboolie's（マウント・バラブーリーズ）:** 転生が必要になる最初の大きな難所。
3. **Strawberry Land & Goat Galaxy（ストロベリーランド＆ゴートギャラクシー）:** より複雑なシークレットユニットが登場する中盤エリア。
4. **Flush Rooms（フラッシュルーム）:** エンドゲームへの入り口となる高難易度ゾーン。
5. **Brainrot Heaven（ブレインロット・ヘブン）:** 最強のボスたちが住む現在の最終ワールド。

## FAQ

**Q: ライブイベント終了後にLuke BlocksやCoral Blocksのユニットを入手できますか？**

A: 現在、ゲーム管理者はこれらのYouTube Brainrotユニットはライブイベント限定であると述べています。イベント期間を逃すと入手は「詰み（cooked）」となり、管理者ティアのユニットはトレードロックされていることが多いため、通常のプレイやトレードでの入手は不可能になる可能性があります。

**Q: トリンケットのグレーディングのためにジェムを稼ぐ最良の方法は何ですか？**

A: 最も効率的な方法は、Trololeroホイールで一度に999枚のチケットを使用することです。これにより通常、数十億のジェムが得られ、それを使ってS+グレードのトリンケットを狙うことができます。

**Q: 最大レベルなのにBrainrot Heavenでダメージが全く通らないのはなぜですか？**

A: Brainrot Heavenのボスは極めて高い防御閾値を持っています。高レベルであっても、Fallen AngelやFrigo Camelloなどのボスに効果的にダメージを与えるには、Sグレードのトリンケット、スーパーノヴァレリック、そして高ティアペットの組み合わせが必要です。

**Q: ライブイベントが開催される特定のワールドはありますか？**

A: メインの戦闘はPota Pain Parkを中心に行われますが、**brain rot evolution ライブイベント**の影響はすべてのワールドに及ぶと予想されます。OceanやDesertのハブには特別なNPCやポータルが出現し、プレイヤーをイベントゾーンへと転送してくれます。

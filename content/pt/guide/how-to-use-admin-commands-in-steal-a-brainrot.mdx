## Detalhamento das Ações de Admin Disponíveis

Os comandos no Steal a Brainrot são categorizados em funções de utilidade e funções de "brainrot" (trollagem). Entender o que cada um faz ajudará você a usá-los de forma mais eficaz durante uma sessão. Embora a lista de comandos possa crescer à medida que os desenvolvedores atualizam o jogo ao longo de 2026, as funções principais continuam sendo as escolhas mais populares para os jogadores.

### Teletransporte (TP)
O teletransporte é o comando de utilidade mais útil. Ele permite mover jogadores pelo mapa instantaneamente. Isso pode ser usado para reunir amigos ou para afastar um jogador problemático de uma área específica. Ao aprender **como usar comandos de admin no Steal a Brainrot**, a função TP costuma ser a primeira com a qual os jogadores experimentam, pois seus efeitos são imediatos.

### Comandos de Troll
A categoria "Troll" é o coração da experiência brainrot. Esses comandos aplicam efeitos visuais ou auditivos a outros jogadores, geralmente envolvendo memes populares ou sons de distração. É importante lembrar que eles foram projetados para diversão e devem ser usados para aumentar a vibração caótica do jogo, em vez de assediar indivíduos.

| Categoria de Comando | Efeitos Comuns | Melhor Caso de Uso |
|---|---|---|
| **TP (Teleporte)** | Move o jogador para sua localização | Organizar grupos ou mover jogadores AFK |

## Gerenciando o Cooldown de 60 Segundos

Para evitar que o jogo se torne impossível de jogar devido ao spam de comandos, os desenvolvedores implementaram um **cooldown (tempo de espera) rigoroso de 60 segundos** em todas as ações de admin. Este é um detalhe crucial a ser mantido em mente ao descobrir **como usar comandos de admin no Steal a Brainrot** de forma eficaz.

Depois de executar um comando em um jogador, você não poderá realizar outra ação de admin até que o cronômetro de um minuto expire. Isso evita que um único admin congele ou teletransporte constantemente a mesma pessoa repetidamente. A estratégia torna-se importante aqui; você precisa decidir se um "troll" específico vale a espera de um minuto antes de poder usar seus poderes novamente.

> **Aviso:** O cooldown se aplica ao administrador, não ao alvo. Se vários admins estiverem no mesmo servidor, um jogador poderá ser alvo de diferentes admins em uma sucessão mais curta.
